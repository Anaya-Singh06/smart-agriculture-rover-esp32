ESP32 Smart Agriculture Rover
Overview

This project is a rover-based embedded system that collects environmental data and provides smart agricultural suggestions.

Features
Multi-point sensing
Soil moisture monitoring
Temperature and humidity sensing
Ultrasonic obstacle detection
Web dashboard
Edge-based decision logic
Hardware Used
ESP32
Soil Moisture Sensor
DHT11 Sensor
Ultrasonic Sensor
Servo Motor
Motor Driver (TB6612FNG)
Working Principle

The rover moves across the field, collects sensor data, processes it on the ESP32, and displays intelligent suggestions on a web dashboard.

Future Work
Automatic irrigation using water pump
Pesticide spraying system
AI-based prediction models
Camera-based crop monitoring
