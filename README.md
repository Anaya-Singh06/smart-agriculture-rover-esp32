ESP32 Smart Agriculture Rover
Overview

This project is a rover-based embedded system that collects environmental data and provides smart agricultural suggestions.

Features:

  Multi-point sensing
  Soil moisture monitoring
  Temperature & humidity
  Obstacle detection
  Web dashboard
  Decision logic

Hardware:

  ESP32
  Soil sensor
  DHT11
  Ultrasonic
  Motor driver

Future Work:

  Automatic irrigation (pump)
  Pesticide spraying
  AI-based prediction
